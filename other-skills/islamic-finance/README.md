# Islamic Finance 伊斯兰金融插件

> 基于 [Panaversity AgentFactory](https://github.com/panaversity/agentfactory-business-plugins) 的伊斯兰金融插件，集成企查查MCP服务，为企业提供符合伊斯兰教法（Shariah）的金融合规和会计解决方案。

---

## 🎯 插件简介

Islamic Finance 是一款专门处理伊斯兰金融合规的智能插件，确保金融产品和服务严格符合伊斯兰教法原则。该插件支持AAOIFI（伊斯兰金融机构会计和审计组织）标准，涵盖伊斯兰银行、伊斯兰债券（Sukuk）、伊斯兰保险（Takaful）等全套伊斯兰金融产品。

**核心理念**：在满足现代金融需求的同时，严格遵守伊斯兰教法原则，服务全球20亿穆斯林人口的金融需求。

## 📋 伊斯兰金融核心原则

### 五大基本原则

| 原则 | 说明 | 与传统金融的区别 |
|------|------|---------------|
| **禁止利息（Riba）** | 严禁任何形式的利息收取和支付 | 存款无利息，贷款无利息 |
| **风险共担** | 盈亏由资金提供方和使用方共同承担 | 银行必须承担交易风险 |
| **禁止过度不确定性（Gharar）** | 禁止投机和过度不确定性的交易 | 合同条款必须明确具体 |
| **资产支持** | 所有交易必须有实物资产或真实服务支撑 | 不能是纯货币借贷 |
| **禁止违禁行业** | 不得投资酒精、猪肉、赌博、色情等行业 | 投资标的需经Shariah审查 |

## 📋 包含功能模块

### 1. Shariah合规审查
- **交易结构审查**：审查金融交易是否符合伊斯兰教法
- **合同条款审核**：确保合同无利息、无过度不确定性
- **投资标的筛选**：筛选符合伊斯兰教法的投资标的
- **Shariah委员会报告**：生成合规审查报告

### 2. 伊斯兰金融工具设计
- **Murabaha（成本加融资）**：商品贸易融资
- **Musharakah（合伙融资）**：共同投资分享盈亏
- **Mudarabah（利润分享）**：资金方出资金，管理方出人力
- **Ijarah（租赁融资）**：资产租赁模式
- **Sukuk（伊斯兰债券）**：资产支持的收益权凭证
- **Takaful（伊斯兰保险）**：互助保险模式

### 3. AAOIFI会计准则
- **财务报表编制**：按AAOIFI标准编制财务报表
- **收益分配计算**：计算伊斯兰金融工具的合规收益
- **准备金计算**：计算符合教法的风险准备金
- **披露要求**：满足AAOIFI披露标准

### 4. 投资组合管理
- **Shariah筛选**：筛选符合教法的上市公司股票
- **净化计算**：计算投资组合中不合规收入的比例
- **Zakat（天课）计算**：计算应缴宗教税
- **投资回报分析**：合规的投资回报分析

### 5. 跨境业务支持
- **多司法管辖区**：支持马来西亚、沙特、阿联酋、卡塔尔等
- **本地合规**：适应各地的伊斯兰金融法规
- **税务优化**：伊斯兰金融工具的税务处理

## 🚀 典型应用场景

### 场景1：Murabaha贸易融资

```
📝 用户需求：
"为企业采购原材料设计Murabaha融资方案，
采购金额1000万，期限6个月"

⚡ 自动执行：
├── 确认Murabaha结构适用性
├── 计算成本加成率
├── 设计付款计划
├── 生成合同模板
├── 计算预期收益
└── Shariah合规审查

📄 输出结果：
    ✅ 融资结构：Murabaha（成本加融资）
    ✅ 采购标的：原材料
    ✅ 融资金额：1000万元
    ✅ 银行成本：950万元（实际采购成本）
    ✅ 利润率：5.26%
    ✅ 客户支付：1000万元（分6期）
    ✅ 银行收益：50万元（合规利润，非利息）
    ✅ 合同条款：符合Shariah要求
    ✅ 风险分担：银行承担商品所有权风险
```

### 场景2：Sukuk伊斯兰债券发行

```
📝 用户需求：
"设计一个5亿元Sukuk发行方案，期限5年，用于基础设施项目"

⚡ 自动执行：
├── 选择Sukuk类型（Ijara Sukuk）
├── 设计资产支持结构
├── 确定租金收益率
├── 计算现金流分布
├── 设计提前赎回机制
├── Shariah合规审查
└── 生成发行文件

📄 输出结果：
    ✅ Sukuk类型：Ijara Sukuk（租赁型）
    ✅ 发行规模：5亿元
    ✅ 基础资产：基础设施项目资产
    ✅ 期限：5年
    ✅ 租金收益率：年租金率6%
    ✅ 现金流：
        - 每年支付租金：3000万元
        - 到期回购：5亿元
    ✅ 投资者回报：6%年化（租金收益，非利息）
    ✅ Shariah合规：已通过委员会审查
    ✅ 评级：AAA（伊斯兰债券评级）
```

### 场景3：投资组合Shariah筛选

```
📝 用户需求：
"对A股投资组合进行Shariah合规筛选，当前持仓50只股票"

⚡ 自动执行：
├── 获取50只股票信息
├── Shariah业务筛选（排除酒精、猪肉、赌博等）
├── 财务比率筛选（资产负债率<33%等）
├── 计算不合规收入比例
├── 计算净化比例
└── 生成合规报告

📄 输出结果：
    ✅ 总持仓：50只股票
    ✅ 合规股票：35只（70%）
    ✅ 不合规股票：15只（需剔除）
    │   ├── 业务不合规：8只（酒类、赌博等）
    │   └── 财务不合规：7只（负债率超标）
    ✅ 净化比例：2.3%
    │   └── 需净化金额：23万元
    │   └── 处置方式：捐赠给慈善机构
    ✅ 调整后组合：
        - 符合Shariah投资标准
        - 预期回报：符合AAOIFI标准
```
