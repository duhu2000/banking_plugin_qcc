# Supply Chain 供应链管理插件

> 基于 [Panaversity AgentFactory](https://github.com/panaversity/agentfactory-business-plugins) 的供应链管理插件，集成企查查MCP服务，为企业提供供应商管理、供应链融资和风险监控的综合解决方案。

---

## 🎯 插件简介

Supply Chain 是一款面向企业采购部门和供应链金融团队的AI智能体插件，覆盖从供应商准入、评估、管理到供应链融资和风险监控的全流程，帮助企业构建稳定、高效、低风险的供应链体系。

**核心理念**：用数据和智能技术管理供应链，降低供应风险，优化资金效率，构建供应链竞争优势。

---

## 📋 包含功能模块

### 1. 供应商管理
- **供应商准入**：管理供应商准入流程
- **资质审核**：审核供应商资质和认证
- **分级分类**：对供应商进行分级分类管理
- **绩效评估**：定期评估供应商绩效
- **关系管理**：维护供应商关系

### 2. 采购管理
- **需求管理**：管理采购需求计划
- **寻源管理**：管理供应商寻源流程
- **比价管理**：自动化比价和议价
- **合同管理**：管理采购合同
- **订单跟踪**：跟踪订单执行情况

### 3. 库存优化
- **需求预测**：预测物料需求
- **安全库存**：计算安全库存水平
- **补货优化**：优化补货时机和数量
- **库存监控**：监控库存健康度
- **呆滞管理**：管理呆滞库存

### 4. 供应链风险管理
- **风险识别**：识别供应链潜在风险
- **风险评估**：评估风险影响和概率
- **风险监控**：持续监控风险信号
- **应急预案**：制定风险应急预案
- **业务连续性**：确保供应链连续性

### 5. 供应链金融
- **供应商融资**：为供应商提供融资方案
- **应收账款管理**：管理应收账款融资
- **保理业务**：管理保理融资
- **动态折扣**：动态折扣管理
- **资金优化**：优化供应链资金效率

        ├── 中风险供应商：增加监控频率
        └── 风险预警：通知采购和风控部门

## 🎬 与企查查MCP的集成价值

### 供应链风险管理（核心应用场景）

| Supply Chain 功能 | 企查查MCP增强 | 业务价值 |
|------------------|--------------|----------|
| **供应商准入** | `qcc-company`: 工商信息核验 | 确保供应商真实存在、经营正常 |
| **资质审查** | `qcc-operation`: 资质证书查询 | 自动验证ISO等认证有效性 |
| **信用评估** | `qcc-risk`: 18类风险信号 | 全面评估供应商信用风险 |
| **风险监控** | `qcc-risk`: 实时风险预警 | 及时发现供应商经营异常 |
| **关联关系** | `qcc-company`: 股权穿透 | 识别供应商之间的隐性关联 |
| **供应链金融** | `qcc-company`: 企业信用等级 | 评估供应商融资资质 |
| **行业分析** | `qcc-operation`: 行业招投标 | 了解供应商市场竞争力 |
