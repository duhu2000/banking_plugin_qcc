## idfa-financial-architect / IDFA财务架构师

**2 个 Skill** — 意图驱动的财务架构工具，自然语言生成专业级可审计财务模型。

| Skill | 功能 |
|-------|------|

**QCC集成**：交易对手信息自动填充、信用风险调整违约概率、行业基准数据引用。

## supply-chain / 供应链管理

**8 个 Skill** — 面向企业采购部门和供应链金融团队，覆盖供应商管理到供应链融资全流程。

| Skill | 功能 |
|-------|------|
| `vendor-assessment` | 供应商 Kraljic 矩阵分类与综合评分（战略/杠杆/瓶颈/常规） |
| `supplier-risk` | 持续供应商风险监控，生成风险简报与分级预警 |
| `spend-analysis` | 采购支出分析、品类分析、尾部支出整合 |
| `invoice-reconciliation` | 三单匹配（采购单/收货单/发票）、AP发票核对 |
| `logistics-brief` | 承运商/货运/物流绩效简报 |
| `supply-chain-brief` | 每周CPO/COO供应链管理仪表盘 |
| `network-design` | 供应链网络设计、仓库选址情景分析 |
| `vendor-communication` | 供应商纠纷函、纠正行动请求（CAR）、沟通模板 |

**QCC集成（核心场景）**：
- 供应商准入：`qcc-company` 工商核验 + `qcc-operation` 资质证书自动验证 + `qcc-risk` 18类风险信号扫描
- 供应链金融白名单：批量筛查供应商，自动准入评级并生成额度建议与风险定价

## 企查查 MCP 集成说明

所有插件均可与企查查 MCP 4个服务器协同工作：

| MCP Server | 别名 | 典型用途 |
|-----------|------|---------|
| `qcc-company` | 企业基座 | 工商核验、股东结构、对外投资、联系方式 |
| `qcc-risk` | 风控大脑 | 失信、被执行、限高、股权冻结等18类风险 |
| `qcc-ipr` | 知产引擎 | 专利、商标、软件著作权、ICP备案 |
| `qcc-operation` | 经营罗盘 | 资质证书、招投标、信用评级、荣誉信息 |
