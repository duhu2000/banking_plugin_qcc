# HR Operations 人力资源运营插件

> 基于 [Panaversity AgentFactory](https://github.com/panaversity/agentfactory-business-plugins) 的人力资源管理插件，集成企查查MCP服务，为企业提供智能化的人力资源全流程管理解决方案。

---

## 🎯 插件简介

HR Operations 是一款面向企业人力资源部门的AI智能体插件，覆盖员工全生命周期管理，从招聘、入职、培训到绩效评估和离职管理，实现HR流程的自动化和智能化。

**核心理念**：让AI成为HR团队的智能助手，自动化处理事务性工作，让HR专注于人才发展和组织建设。

---

## 📋 包含功能模块

### 1. 智能招聘管理
- **简历智能筛选**：基于岗位要求自动筛选和评分简历
- **面试智能安排**：自动协调面试官和候选人时间
- **人才库管理**：自动归档和分类候选人信息
- **Offer智能生成**：根据候选人信息自动生成Offer Letter

### 2. 员工入职管理
- **入职流程自动化**：自动生成入职任务清单和日程安排
- **文档智能准备**：自动生成和收集入职所需文档
- **培训计划制定**：根据岗位自动制定新员工培训计划
- **系统账号开通**：自动协调IT开通邮箱、系统等账号

### 3. 考勤与假期管理
- **智能考勤分析**：自动分析考勤异常并生成报告
- **假期余额管理**：实时计算和管理员工假期余额
- **排班智能优化**：基于业务需求优化员工排班
- **加班自动计算**：自动计算加班时长和费用

### 4. 绩效管理
- **绩效目标制定**：辅助制定SMART绩效目标
- **360度评估**：自动化360度绩效评估流程
- **绩效数据分析**：分析绩效数据，识别高潜人才
- **改进计划生成**：基于绩效结果生成绩效改进计划

### 5. 员工培训发展
- **培训需求分析**：基于岗位和能力差距分析培训需求
- **培训计划管理**：制定和管理年度培训计划
- **学习路径推荐**：为员工推荐个性化学习路径
- **培训效果评估**：跟踪和评估培训效果

### 6. 员工关系管理
- **员工满意度调研**：自动化员工满意度调研和分析
- **离职面谈管理**：标准化离职面谈流程和记录
- **员工关怀提醒**：生日、入职纪念日等自动提醒
- **内部沟通优化**：优化内部信息传递和沟通效率

## 🚀 典型应用场景

### 场景1：智能招聘流程

```
📝 用户需求：
"收到100份JAVA工程师简历，帮我筛选出符合要求的候选人"

⚡ 自动执行：
├── 提取岗位要求：3年经验、Spring Boot、微服务
├── 自动解析100份简历
├── 按匹配度评分排序
├── 筛选出前20名候选人
├── 自动发送面试邀请邮件
└── 生成面试评估表

📄 输出结果：
    ✅ 简历解析完成：100份
    ✅ 匹配度≥80%：25人
    ✅ 面试邀请已发送：20人
    ✅ 自动排期完成：15人已确认面试时间
    ✅ 面试官日程已同步
```

### 场景2：新员工入职自动化

```
📝 用户需求：
"张三明天入职，帮我准备入职流程"

⚡ 自动执行：
├── 生成入职任务清单（17项）
├── 准备入职文档包
├── 预订入职培训会议室
├── 通知IT准备电脑和账号
├── 通知行政准备工位和门禁
├── 发送欢迎邮件给团队
└── 设置30天试用期跟踪计划

📄 输出结果：
    ✅ 入职日：2026-04-01
    ✅ 入职文档：已准备（合同、保密协议等）
    ✅ 培训安排：9:00-12:00 入职培训
    ✅ IT准备：笔记本电脑、邮箱、系统账号
    ✅ 行政准备：工位、门禁卡、办公用品
    ✅ 团队欢迎：邮件已发送
```

### 场景3：绩效评估自动化

```
📝 用户需求：
"启动Q1季度绩效评估，评估范围：技术部50人"

⚡ 自动执行：
├── 生成绩效评估通知
├── 自动分配评估人（上级、同事、下属）
├── 发送评估问卷（360度）
├── 设置评估截止日期提醒
├── 收集评估结果
├── 生成绩效分析报告
└── 安排绩效面谈日程

📄 输出结果：
    ✅ 评估人员：50人
    ✅ 评估问卷：已发送150份
    ✅ 完成进度：85%（128/150）
    ✅ 绩效分布：A级15%、B级60%、C级20%、D级5%
    ✅ 高潜人才识别：8人
    ✅ 需改进人员：10人
    ✅ 面谈日程：已安排
```

## 🎬 与企查查MCP的潜在集成

### 背景调查与合规管理

| HR Operations 功能 | 企查查MCP增强 | 应用场景 |
|-------------------|--------------|----------|
| 高管背景调查 | `qcc-company`: 查询候选人关联企业 | 招聘高管时核实候选人商业背景 |
