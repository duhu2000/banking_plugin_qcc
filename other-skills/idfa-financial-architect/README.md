# IDFA Financial Architect 财务架构师插件

> 基于 [Panaversity AgentFactory](https://github.com/panaversity/agentfactory-business-plugins) 的财务建模插件，集成企查查MCP服务，为企业提供意图驱动的财务架构解决方案。

---

## 🎯 插件简介

IDFA（Intent-Driven Financial Architecture，意图驱动的财务架构）是一款革命性的财务建模工具。用户只需用自然语言描述业务场景，AI即可自动生成专业级、可审计的财务模型。

**核心理念**：让非财务专业人员也能快速构建复杂财务模型，实现"说人话，出模型"。

---

## 📋 核心架构

### 三层架构设计

```
┌─────────────────────────────────────┐
│          输出层 (Output)             │
│    财务报表、图表、分析结论          │
├─────────────────────────────────────┤
│          计算层 (Calculation)        │
│    公式、逻辑、数据运算              │
├─────────────────────────────────────┤
│          输入层 (Input)              │
│    假设、参数、基础数据              │
└─────────────────────────────────────┘
```

### 四重护栏机制

| 护栏 | 功能 | 防护内容 |
|------|------|----------|
| **护栏1** | 循环引用检测 | 防止公式循环引用导致模型错误 |
| **护栏2** | 硬编码检查 | 防止手动输入数值，确保数据可追溯 |
| **护栏3** | 结构合规性 | 确保三层架构不被破坏 |
| **护栏4** | 审计链完整 | 确保每个数字都有来源和路径 |

## 📋 包含功能模块

### 1. 自然语言建模
- **意图理解**：解析用户的自然语言描述，提取建模需求
- **智能映射**：将业务场景映射到标准财务模型模板
- **自动构建**：自动生成符合三层架构的财务模型

### 2. 命名范围管理
- **智能命名**：用有意义的名称替代单元格地址（如`Revenue_Y1`替代`B5`）
- **名称导航**：通过名称快速定位和修改模型
- **公式可读性**：公式使用名称而非地址，提高可读性

### 3. 模型模板库
- **DCF估值模型**：现金流折现估值
- **LBO杠杆收购模型**：并购融资分析
- **M&A并购模型**：并购协同效应分析
- **房地产模型**：开发项目投资分析
- **初创企业模型**：融资轮次估值
- **预算规划模型**：年度财务预算

### 4. 审计友好设计
- **公式透明**：每个计算都有清晰公式
- **数据溯源**：每个数字都可追溯到源头
- **变更记录**：记录模型修改历史
- **版本控制**：支持模型版本管理

### 5. 情景分析
- **基础情景**：标准假设下的预测
- **乐观情景**：最佳情况分析
- **悲观情景**：最坏情况分析
- **敏感性分析**：关键变量影响分析

## 🚀 典型应用场景

### 场景1：房地产项目投资分析

```
📝 用户需求：
"我要建一个房地产项目的现金流模型，
地价5亿，建设期3年，销售从第2年开始，
单价3万/平米，共10万平米"

⚡ 自动执行：
├── 识别项目类型：房地产开发
├── 提取关键参数：
│   ├── 地价：5亿
│   ├── 建设期：3年
│   ├── 销售期：第2年开始
│   ├── 单价：3万/平米
│   └── 总面积：10万平米
├── 加载房地产模型模板
├── 构建三层架构：
│   ├── 输入层：所有假设参数
│   ├── 计算层：成本分摊、收入预测、现金流
│   └── 输出层：IRR、NPV、投资回收期
├── 应用四重护栏验证
└── 生成Excel模型文件

📄 输出结果：
    ✅ 模型类型：房地产投资模型
    ✅ 输入参数：23项
    ✅ 计算公式：156个
    ✅ 输出指标：
        - 项目总投资：8.5亿元
        - 预计销售收入：30亿元
        - 项目IRR：18.5%
        - 投资回收期：3.2年
        - NPV（10%折现率）：12.3亿元
    ✅ 敏感性分析：房价±10%对IRR的影响
    ✅ 审计追踪：完整
```

### 场景2：初创企业估值

```
📝 用户需求：
"帮我做一个B轮科技公司的DCF估值模型，
今年收入5000万，年增长率50%，
预计3年后IPO，目标估值50亿"

⚡ 自动执行：
├── 识别模型类型：初创企业DCF估值
├── 提取关键参数：
│   ├── 当前收入：5000万
│   ├── 增长率：50%
│   ├── 融资轮次：B轮
│   ├── 目标退出：IPO
│   └── 目标估值：50亿
├── 加载初创企业估值模板
├── 构建预测模型（5年）
├── 计算WACC和终值
├── 生成估值区间分析
└── 输出投资建议

📄 输出结果：
    ✅ 估值模型：DCF三阶段模型
    ✅ 收入预测：5000万→3.38亿（5年）
    ✅ 估值结果：
        - 保守情景：35亿元
        - 基础情景：48亿元
        - 乐观情景：65亿元
    ✅ 隐含IRR：B轮投资者45%
    ✅ 关键假设：退出PE倍数25x
    ✅ 风险因素：增长率敏感性±20%
```

### 场景3：并购交易模型

```
📝 用户需求：
"做一个收购交易模型，
目标公司EBITDA 2亿，收购价20亿，
其中债务融资12亿，股权融资8亿"

⚡ 自动执行：
├── 识别模型类型：LBO杠杆收购
├── 提取交易结构：
│   ├── 目标EBITDA：2亿
│   ├── EV/EBITDA：10x
│   ├── 总对价：20亿
│   ├── 债务融资：12亿（60%）
│   └── 股权融资：8亿（40%）
├── 构建Sources & Uses表
├── 模拟资本结构变化
├── 计算杠杆率和利息覆盖
├── 预测退出回报
└── 生成投资回报分析

📄 输出结果：
    ✅ 交易结构：
        - 企业价值：20亿元
        - 债务：12亿元（利率6%）
        - 股权：8亿元
    ✅ 财务指标：
        - 杠杆率：6.0x EBITDA
        - 利息覆盖：2.5x
    ✅ 投资回报（5年退出）：
        - 退出EBITDA：2.8亿
        - 退出估值：28亿
        - 股权回报：2.5x
        - IRR：20%
    ✅ 敏感性：EBITDA增长对回报的影响
```

## 🎬 与企查查MCP的潜在集成

### 财务模型数据增强

| IDFA 功能 | 企查查MCP增强 | 应用场景 |
|----------|--------------|----------|
| 交易对手信息 | `qcc-company`: 自动填充企业工商信息 | 模型中涉及的企业自动获取背景信息 |
| 信用风险评估 | `qcc-risk`: 企业信用等级和风险信号 | 调整模型的违约概率假设 |
| 行业基准数据 | `qcc-operation`: 行业招投标、资质数据 | 提供行业对比和基准分析 |
| 关联风险 | `qcc-company`: 股权穿透和关联方 | 识别模型中的关联交易风险 |
| 投资标的筛选 | `qcc-operation`: 企业信用评级 | 筛选适合投资的优质企业 |
