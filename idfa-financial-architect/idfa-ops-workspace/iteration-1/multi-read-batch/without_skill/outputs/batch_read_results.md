# Year 1 Financial Results

**Source:** `ops_test_model.xlsx`

## Assumptions

| Parameter             | Value   |
|-----------------------|---------|
| Unit Price            | $100    |
| Units Sold (Year 1)  | 50,000  |
| Variable Cost / Unit  | $40     |
| Fixed Costs           | $1,000,000 |
| Tax Rate              | 30%     |

## Year 1 Income Statement

| Metric              |        Amount |
|----------------------|--------------:|
| Units Sold           |        50,000 |
| Revenue              |  $5,000,000   |
| Variable Costs       | ($2,000,000)  |
| **Contribution Margin** | **$3,000,000** |
| Fixed Costs          | ($1,000,000)  |
| **EBIT**             | **$2,000,000** |
| Tax (30%)            |   ($600,000)  |
| **Net Income**       | **$1,400,000** |

## Derivation Notes

All values computed from the Assumptions sheet using the formula chain in the Calculations sheet:

- Revenue = Units Sold x Unit Price = 50,000 x $100
- Variable Costs = Units Sold x Variable Cost/Unit = 50,000 x $40
- Contribution Margin = Revenue - Variable Costs
- EBIT = Contribution Margin - Fixed Costs
- Tax = EBIT x Tax Rate
- Net Income = EBIT - Tax
