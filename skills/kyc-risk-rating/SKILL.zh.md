## 🎯 风险为本方法原则

FATF风险为本方法要求AML/CFT措施与识别的风险成比例。高风险客户接受增强尽职调查（EDD）；低风险客户在某些情况下可能接受简化尽职调查（SDD）。

每个客户必须被分配风险评级，且评级必须定期审查。

---

## 📊 风险评分框架 — 四个维度

### 维度1：客户类型风险

| 客户类别 | 风险评分 |
|----------|----------|
| 上市公司（主要交易所） | 1 — 低 |
| 受监管金融机构（本国） | 2 — 低 |
| 本国企业（私营，无PEP关联） | 3 — 中等 |
| 高净值个人 | 3 — 中等 |
| 非营利组织/慈善机构 | 4 — 中高 |
| 外国私营企业 | 4 — 中高 |
| 离岸结构（开曼、BVI、巴拿马） | 5 — 高 |
| 信托或基金会（复杂受益人结构） | 5 — 高 |
| PEP个人（一级 — 外国） | 5 — 高（强制EDD） |
| 现金密集型业务（珠宝商、货币兑换、赌场） | 5 — 高 |

---

### 维度2：地理风险

| 地理区域 | 风险评分 |
|----------|----------|
| FATF成员，低TI-CPI风险 | 1 — 低 |
| FATF成员，中等TI-CPI风险 | 2 — 低-中 |
| FATF加强跟进（灰名单） | 4 — 高 |
| FATF黑名单司法管辖区 | 5 — 非常高 |
| 非FATF司法管辖区，强AML制度 | 3 — 中等 |
| 非FATF司法管辖区，弱AML制度 | 4 — 高 |

> 📌 检查 fatf-gafi.org 的当前FATF灰/黑名单 — 每年更新三次。
> 透明国际清廉指数（CPI）：低于40/100的分数表示高腐败风险。

---

### 维度3：产品/服务风险

| 产品/服务 | 风险评分 |
|-----------|----------|
| 基本活期账户（本国客户） | 1 — 低 |
| 定期储蓄/存款 | 1 — 低 |
| 零售抵押贷款 | 2 — 低-中 |
| 企业活期账户 | 3 — 中等 |
| 国际电汇 | 4 — 中高 |
| 私人银行/财富管理 | 4 — 中高 |
| 代理银行业务 | 5 — 高 |
| 贸易融资 | 5 — 高 |
| 加密货币相关服务 | 5 — 高 |
| 现金密集型交易 | 5 — 高 |

---

### 维度4：关系/行为风险

| 指标 | 风险评分 |
|------|----------|
| 长期客户，行为一致 | 1 — 低 |
| 新客户，无先前关系 | 3 — 中等 |
| 复杂或不一致的业务解释 | 4 — 中高 |
| 不愿提供CDD文件 | 5 — 高 |
| 无独立核验的第三方介绍 | 4 — 中高 |
| 不利媒体（未核实） | 4 — 中高 |
| 不利媒体（已核实/刑事定罪） | 5 — 高 |
| 此前对该客户的SAR（本行或其他FI） | 5 — 高 |

## 🧮 综合风险评级计算

综合评分 = 四个维度评分的加权平均：
- 客户类型：35%
- 地理：30%
- 产品/服务：20%
- 关系/行为：15%

| 综合评分 | 整体风险评级 | CDD级别 | 监控频率 |
|----------|-------------|---------|----------|
| 1.0 - 2.0 | 低 | 标准CDD | 每5年 |
| 2.1 - 3.0 | 中等 | 标准CDD | 每3年 |
| 3.1 - 4.0 | 高 | 增强CDD | 每年 |
| 4.1 - 5.0 | 非常高 | 增强CDD + 高级管理层批准 | 每6个月或更频繁 |

### 计算方法示例

客户类型：外国私营企业 = 4
地理：FATF灰名单司法管辖区 = 4
产品：代理银行业务 = 5
行为：新客户 = 3

综合 = (4×0.35) + (4×0.30) + (5×0.20) + (3×0.15) = 4.05 = **非常高**

## ⚠️ 强制覆盖（无论评分如何，自动评为高/非常高）

以下情况自动将客户分类为高风险或非常高风险：

- PEP身份（任何级别）→ 非常高（强制EDD、高级管理层批准）
- FATF黑名单司法管辖区客户 → 非常高
- 超过定义阈值的现金密集型业务 → 高
- 受执法机关要求或已知调查的客户 → 非常高
- 受益所有人结构包括无受益所有人登记册的司法管辖区 → 高

## 🔄 KYC更新触发条件（定期计划之外）

当以下情况发生时，触发计划外的KYC更新：

- 客户或关联方的不利媒体警报
- 所有权或受益所有人变更
- 交易行为显著变化
- 执法机关联系或信息要求
- 客户通知重大变更（新业务、新地址、董事变更）
- 对本客户提交内部SAR
- 客户被交易监控系统加入观察名单

## ⚠️ 禁止事项

- **绝不要**在不检查所有四个维度的情况下分配风险评级 —— 遗漏维度（特别是地理或行为）会系统性低估风险，无法通过监管审查
- **绝不要**在没有记录的高级管理层批准和明确监管依据的情况下向下覆盖强制高/非常高分类 —— PEP身份和FATF黑名单触发因素不是自由裁量的
- **绝不要**在存在强制覆盖触发因素时仅依赖综合评分 —— 无论加权平均值如何，覆盖都优先
- **绝不要**在触发事件发生时推迟KYC更新 —— 基于触发的更新是监管期望，延迟会造成合规缺口，监管会将其视为发现项
